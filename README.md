# Sport Map

## How to install
- Clone or download project
- Open the project in Visual Studio Code

## How to run project
- Write or copy in terminal
``` bash
npm install
```
- Write or copy in terminal
``` bash
npm run dev
```
- Control + left click on the local link
- Login page will appear

### How to register
- Click **Register** link in the navigation menu on top of the page
- Enter any email that contains this symbol @
- Enter name that less than 128 symbols
- Enter last name that less than 128 symbols
- Enter password that has atlest 6 symbols and not more than 100 symbols and contains 1 non-alphanumeric character
- Repeat password
- Click **Submit**
- Home Page will appear

## GPS SEssions Page
- Show all user`s maps
- User can filter map list
- User can view map of other user

## User`s GPS Sessions Page

Show all your maps, which you can interact

### Create Map
- Click **Create New**
- Enter name that must contains atleast 2 symbols and less than 257
- Enter description that must contains atleast 2 symbols and less than 4097
- Click on the field below **Session Type** to choose it
- Enter Pace Min that must be equally or more than 60
- Enter Pace Max that must be more than Pace Min
- Click **Create**

### Map Interaction
- Click **Map view**
- Map page will appear
- Can scroll up or down to make map bigger or smaller
- Click on the map to add marker
- Click another one to create one more marker
- Now appear route first marker is the start and final added marker is end
- Click on the marker to delete it
- Drag marker to update it location

### Update Map
- Click **Update**
- Modify fields
- Click **Edit*

### Delete Map
- Click **Delete**
- Notification will appear
- Click **OK**
